◆ afe_get_buffer_callback

cy_afe_get_output_buffer_callback_t cy_afe_config_t::afe_get_buffer_callback

Register callbacks to get the buffer to fill the output data.

User may choose to register the callback to provide the memory for the output buffer. If callback is registered, then AFE middleware will invoke the callback to get memory for output buffer. And ownership of freeing the buffer is with the application. If callback is not registered, AFE middleware will have the default implementation of callback through which static buffer memory will be used, output data will be filled and provided to application through cy_afe_output_callback_t callback. Note that, static buffer memory will be used in subsequent output callbacks and user doesn't need to free the memory.
